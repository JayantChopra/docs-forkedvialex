---
title: "Quickstart"
description: "Install the GitHub app, connect repos, pick a model — Polarity analyzes every commit and opens PRs."
---

<Info>
  Polarity is dashboard‑only — there is no API or SDK to set up.
  Install the GitHub App, connect your repos, choose a model, and you're done.
</Info>

<Steps>
<Step title="Log into the dashboard">
  Sign in with your work email. Invite teammates to collaborate.
</Step>

<Step title="Add orgs and repos">
  Install the Polarity GitHub App and connect GitHub, GitLab, or self‑hosted.
  Select which repositories to analyze.
</Step>

<Step title="Choose the model">
  Use Polarity defaults or bring your own. Override per repo or branch.
</Step>

<Step title="Polarity analyzes every commit">
  We simulate behavior, trace dependencies, and flag regressions proactively.
</Step>

<Step title="Polarity publishes PRs">
  We open high‑quality PRs that improve reliability, security, and performance.
</Step>

<Step title="Learns over time — or lock defaults">
  Approvals and comments refine future PRs. Prefer deterministic behavior? Toggle RL off.
</Step>
</Steps>
